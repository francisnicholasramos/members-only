# Members Only 

Send message anonymously 

Live: https://members-only-jlk8.onrender.com

## Features
- Authentication — Secure login and signup using Passport.js with bcrypt password hashing.
- Anonymous Messaging — Regular members can send messages without revealing their identity.
- Temporary Access Pass — Non-admin users can enter a secret code to temporarily reveal the identity of message senders.

## Tech Stack and Library used
Typescript
ExpressJS
PassportJs (authentication)
Supabase
EJS (Embedded Javascript) 
Bcrypt (for hashing)
Token handling (jwt)
